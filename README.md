# Stellar Population Synthesis Modeling of Galaxy Spectra

An interactive walk-through Jupyter Notebook that provides an introduction to how stellar population synthesis modeling is used to fit galaxy spectra. These concepts are applicable to understanding how modern SED fitting softwares work!

## Structure

- `stellar_population_modeling_of_a_galaxy.ipynb` - walk-through notebook
- `plots/` - example outputs from the notebook
- `data/` - data required for the notebook

## Setup

Make sure you have the following Python packages installed:
- numpy
- matplotlib.pyplot
- pandas

using `conda install [package name]` or `pip install [package name]`

## Usage

Download the files into a directory on your machine and start going through the Jupyter Notebook!
